1. System Requirements:
   a. PHP Versi 7.4 keatas
   b. Ekstensi intl di-enabled kan
   c. Ekstensi mbstring di-enabled kan
   d. Ekstensi json di-enabled kan
2. Buat database spkambing
3. Impor file spkambing.sql
4. Ekstrak file spkambing.zip ke htdocs
5. Instalasi
   a. Sesuaikan $baseURL app\Config\App.php
   b. Sesuaikan konfigurasi app\Config\Database.php
